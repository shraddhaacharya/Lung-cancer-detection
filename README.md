Software required :Python version 3.8.5 and above
Install the Python

  pip instll Pillow
  pip install imutils
  pip install tensorflow==2.4.0
  pip install keras==2.4.0
  pip install sklearn
  pip install matplotlib
Using the pythonIDE open the main.py file
run the main.py
